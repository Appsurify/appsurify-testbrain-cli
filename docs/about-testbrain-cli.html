About TestbrainCLI

For interactive use, you first launch a shell such as cmd.exe on Windows, or Bash on Linux or macOS, and then issue a command at the shell prompt. To automate repetitive tasks, you assemble the CLI commands into a shell script using the script syntax of your chosen shell, and then you execute the script. You can install the Testbrain CLI locally on Linux, Mac, or Windows computers. It can also be used from inside a Docker container.

Current version

The current version of the Appsurify Testbrain CLI is 2023.11.9. To find your installed version and see if you need to update, run testbrain --version.

Run Testbrain CLI in your CI/CD pipeline or locally

Testbrain not provides native solutions for Azure Pipelines, CircleCI, GitHub, and TeamCity. But for CI/CD systems, you can use Testbrain CLI from PyPi package or Docker images. To run Testbrain CLI locally, you can use Python interpreter or Docker images.
